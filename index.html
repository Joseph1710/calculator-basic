In designing inputs, i just only set the border to 0, and added a bottom-border which is 1px solid and a color of #ccc. And I also added a 10px padding, in order for it to expand. And for the text to align right just set the text-alignment to 'right'

input functionality

To give a functionality to an input you need the onkeyup listener to get and process every inputs of your user. What I did is, first you need to get the value of your input element, and get the users entered value, but don't process it directly. For example if the user entered value is just a plain/simple expression who has only one(1) operator for ex. 1+1. you don't need to process it directly, instead you let your user 'click' the equal button. else if the user entered an expression like this '1+1+' that's the time that you will process it directly in the input listener. For me you can have any style to satisfy or process your user inputs. I implemented it for a reason because, I don't want my user enter/input multiple operator.We can still implement that, but for now we'll just try a simple with a little tricky or hard style.

button numbers

The button action listener was also simple you just need to concatenate all your users inputs every, time they will click a button numbers. for ex. if a user click a button number 1 and 2. You just need to assign it to your GLOBAL variable, and don't forget to set also your input element's value based on the user button clicked.

case 1 :

USER_PROCESSES += 1

entry.value = USER_PROCESSES

break ;

case 2 :

USER_PROCESSES += 2

entry.value = USER_PROCESSES

break ;

equal buttons

The equal buttons is the main focus of this calculator because it process all users input. And to get the results for every opertions that your users clicked. first you need to get the value of your input element and loop on that data, and the second is you need to extract the numbers and operators and store it to a different variables. Theres so many way on how you can do that, whether you will use an array or just a simple string variables that is easy to manipulate and access, well it's up to you on how to do that. Just focus on the goal on how to extract and manipulate your data!
